# Glossary

| Term | Working definition from this guide |
| --- | --- |
| Alternate keying | A deliberate change to connector polarization so similar connectors cannot cross-mate |
| Backshell | Rear accessory that handles strain relief, shield termination, sealing, and cable exit angle |
| Blind mate | A connector engagement where the operator cannot directly see the interface during mating |
| Bundle derating | Reducing allowed current because multiple loaded contacts heat each other |
| Cable gland | The sealing/clamping feature that grips the cable jacket at the rear of the connector |
| Configuration control | Treating the full connector interface as controlled design hardware, not an interchangeable convenience item |
| Contact size | The physical contact size that drives wire-gauge compatibility and current capacity |
| Creepage | Surface distance across insulation that matters for voltage withstand |
| Crimp contact | A contact terminated by compressing wire into a barrel with a calibrated die |
| Dielectric withstand | The hi-pot test voltage the connector survives without breakdown |
| Dummy plug | A plug used to occupy unused ports and preserve sealing/configuration |
| Dust cap | A protective cap used on unmated connectors to reduce dirt, water, and pin damage |
| Field-wireable | A connector intended to be assembled onto cable in the field rather than factory molded |
| Flange mount | A panel-mount style that uses perimeter fasteners for rigid alignment and gasket control |
| Fretting corrosion | Oxidation caused by small repeated motion at the contact interface |
| Ground-first sequencing | A mating sequence where ground mates first and breaks last |
| Hermetic | A sealed construction intended to block leakage through the connector body itself |
| Hybrid connector | A connector arrangement that combines different media such as power, signal, data, or coax in one shell |
| Insert arrangement | The exact contact layout, sizes, and positions inside a connector insert |
| Jam-nut mount | A compact circular panel mount retained by a rear nut through a round panel cutout |
| Keying / polarization | The physical anti-mis-mate geometry that controls orientation and mating compatibility |
| Mating cycles | The rated number of connect/disconnect cycles before the connector is considered worn |
| Pin / socket | Electrical contact gender; separate from plug/receptacle body style |
| Plug / receptacle | Mechanical body style; separate from the contact gender |
| QPL | Qualified Products List; used when qualification or approved-source status matters |
| Scoop-proof | A connector feature that protects contacts during mating under harsh handling conditions |
| Service class | The combination of shell material, finish, and environmental intent used in a connector family |
| Strain relief | The mechanical feature that transfers cable load into the connector body instead of the contacts |
| TPA | Terminal position assurance; a secondary feature that helps prevent contact pushback |
| Wire seal | A per-contact seal that closes around the wire jacket and prevents moisture tracking into the connector body |
