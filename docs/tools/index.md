# Tools & Templates

Reusable engineering-document templates derived from the guide. Copy them into your own design package and fill in the project-specific values. They are deliberately generic — no ratings or part numbers are assumed for you.

| Template | Use it to | Based on |
|---|---|---|
| [Connector Selection Checklist](connector-selection-checklist.md) | Walk an interface from definition to documentation without skipping a step | [Practical Checklist](../10-selection-checklist.md) |
| [Connector Comparison Matrix](connector-comparison-matrix.md) | Score candidate families side by side before you commit | [Exercise 2](../13-hands-on-exercises.md) |
| [Cable Drawing Template](cable-drawing-template.md) | Turn a pinout into a buildable cable drawing | [Exercise 6](../13-hands-on-exercises.md) |
| [Connector ICD Template](connector-icd-template.md) | Define an interface so procurement and manufacturing can build it | [Exercise 7](../13-hands-on-exercises.md) |
| [Design Review Checklist](design-review-checklist.md) | Run a connector risk pass before a design review | [Sections 10–11](../11-red-flags.md) |

:::note

These are working templates, not standards. Adapt them to your program, and remember that the datasheet, applicable standard, and customer/program requirements always win.

:::
