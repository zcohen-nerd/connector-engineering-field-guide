# Example: Rugged Control Box

This example is derived from [Exercise 1](../13-hands-on-exercises.md) and stays at the family-selection / documentation level. Project-specific values are intentionally left as `TBD`. The point is the *structure* of a complete interface definition, not the numbers.

The enclosure is a small sealed control box with these external interfaces: 24 VDC input, Ethernet, CAN, four sensors, one motor output, and one debug/service port. For selection reasoning on each of these, see the matching [decision paths](../decision-paths/index.md).
